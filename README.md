## Visão Geral

Este repositório contém o **site de documentação oficial** do projeto **Lynx Assistant**.  
Ele inclui:

- Página estática em HTML/CSS/JS  
- Estrutura de componentes e demonstrações  
- Interface de preview e dashboard  
- Guia visual das funcionalidades  
- Documentação pública do sistema

Este repositório não contém o código-fonte do assistente em Python.  
O código principal se encontra no repositório:

https://github.com/gabriel-araujo-git/Lynx-Assistant

## Executando Localmente

Como é um projeto estático, basta clonar o repositório:

```bash
git clone https://github.com/gabriel-araujo-git/Lynx-Assistant-Docs.git
cd Lynx-Assistant-Docs
```

E abrir o arquivo:

```
index.html
```

em qualquer navegador moderno.

## Publicação

Este projeto pode ser hospedado em:

- GitHub Pages  
- Netlify  
- Vercel  
- Qualquer servidor de arquivos estáticos  

Para publicar via GitHub Pages:

1. Vá em *Settings*  
2. Acesse *Pages*  
3. Configure como:  
   `Deploy from branch`  
4. Branch: `main`  
5. Pasta: `/root`
